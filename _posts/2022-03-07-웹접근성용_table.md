# 웹 접근성 표준이란?
- 웹 접근성 표준이란 **장애인이 비장애인과 동등하게 웹 콘텐츠에 접근할 수 있도록 웹 콘텐츠를 제작하는 방법에 관하여 기술하고 있다.** 라고 나와있습니다.   
(참고문헌 : 한국형 웹 콘텐츠 접근성 지침 2.1 중)
