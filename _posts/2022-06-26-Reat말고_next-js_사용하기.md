# next.js 프로젝트 생성하기
- next.js 를 생성하는법은 React 프로젝트를 생성할 때 처럼 매우 간단합니다.

```console
    // React

    // use npm
    npm create-react-app 프로젝트명

    // use yarn
    yarn create react-app 프로젝트명

    // next.js

    // npm
    npx create-next-app

    // yarn
    yarn create next-app
```

- 일반적인 React와 다를 것이 없습니다....

# next.js 개발서버 구동하기
- next.js 개발서버 구동도 어렵지 않습니다.

```console
    // React

    // use npm
    npm start

    // use yarn
    yarn start

    // next.js

    // use npm
    npm run dev

    // use yarn
    yarn dev
```
